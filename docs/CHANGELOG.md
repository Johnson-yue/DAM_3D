# DAM 3D Render 更新日志

## v0.1 — 2026-07-19

> 修复显示与 BLEND 格式支持（资产库交互、预览批注、FBX/OBJ/BLEND 材质与取景、检索 Debug）。

### UI / 资产库

1. **点击画面进入预览**：资产卡片缩略图（黑色背景预览区域）支持鼠标左键点击，直接进入预览界面；移除原先距离较远的「预览 / 批注」文字链接。
2. **移除重处理**：资产库卡片去掉「重处理」按钮；后端不再提供入库后的重处理入口。原则：资产入库并完成预处理后不可再触发重处理。
3. **卡片底部展示标签**：原「预览 / 批注」与「重处理」所在行改为展示资产标签，最多显示前 3 个。
4. **标签可改可删**：资产库卡片上的标签支持就地修改与删除；每次操作后立即写入托管库 SQLite，刷新后保持一致。预览页侧栏标签交互同步支持单标签删除与保存持久化。

### 批注编辑（预览页）

5. **可进入 / 退出编辑态**：选中「色框 / 箭头 / 文字 / 选择」进入批注编辑；点「旋转」、按 `Esc`，或「保存批注」成功后自动退出编辑并恢复 3D 旋转/缩放。
6. **批注增删改**：
   - 增加：色框拖拽绘制、箭头拖拽、文字单击落点；
   - 修改：在「选择」模式下可移动批注；色框拖角点调整大小；箭头拖端点改方向；双击文字修改内容；
   - 删除：选中后点「删除选中」，或按 `Delete` / `Backspace`。
7. **修复**：此前进入色框等工具后无法回到 3D 交互，即使保存批注后仍卡住的问题。

### FBX / OBJ 原材质与资产库一致

8. **默认材质判定**：预览交互时，对 FBX/OBJ 先检测导出 GLB 是否含贴图或非中性颜色等原材质；有则默认「原材质」，无则默认「灰模」。GLB/GLTF 仍默认原材质。
9. **固定图同步规则**：Blender 生成资产库/预览首帧时，对 FBX/OBJ 同样先检测原材质——有则 EEVEE 彩色渲染，无则灰模。此前 FBX/OBJ 固定图一律灰模，导致库卡片与点「原材质」后的交互预览不一致。
10. **存量刷新**：提供 `backend/scripts/refresh_fbx_obj_views.py`，用于按新规则重渲库内已有 FBX/OBJ 固定图并重建 embedding。

### 资产删除

11. **单删**：资产卡片缩略图右上角删除按钮；确认后硬删除（库记录、托管原件副本、GLB/多视图、批注、faiss 向量）；源盘不动。
12. **多选批量删**：卡片左上角常显勾选框；选中 ≥1 个时工具栏出现「删除选中 (N)」；确认后调用批量删除。
13. **API**：`POST /api/assets/bulk-delete`，body `{ ids: number[] }`，返回 `deleted` / `failed`。
14. **多选状态保持**：勾选存于 SPA 内存 store；进入预览再返回资产库仍保留勾选；刷新页面清空。仅在取消勾选、删除成功或资产已不存在时清理。

### BLEND 主体筛选修复

15. **根因**：旧逻辑按「面数 ≥ 最大面数 30%」删网格；高面数辅助球会留下，魔方等小零件被删，固定图只剩球。
16. **新规则**：BLEND 保留全部有效 MESH；仅删除明显背景平面（名称含 plane/平面，或极薄且少面）；不再按面数删除零件。OBJ/FBX/GLTF/GLB 不变。
17. **存量刷新**：`backend/scripts/refresh_blend_views.py` 重处理库内已有 `.blend`。

### BLEND 材质与取景（玫瑰等）

18. **原材质默认**：BLEND 与 FBX/OBJ 对齐——检测到原材质则固定图与预览默认「原材质」，否则灰模。
19. **取景修复**：优先跳过隐藏网格；用主体簇尺寸/距离忽略过大或远离的异常体。名称含 `plane`/`平面` 的网格不再一律剔除（见下条细化）。
20. **色彩管理兼容**：原材质渲染的 View Look 优先使用 Blender 5 可用的 `Medium High Contrast`（不再写死 `AgX - …` 前缀），避免枚举不存在导致 `.blend` 预处理失败。
21. **全量刷新**：再次运行 `refresh_blend_views.py` 更新库内全部 `.blend`。

### BLEND 结构保真（椰子树等）

22. **原则**：预处理不得改变物体结构——`.blend` 禁止 `transform_apply`；不隐藏 CURVE / EMPTY / ARMATURE 等修饰器与约束依赖。
23. **根因**：椰子树树干依赖 `ARRAY+CURVE`，叶子含 `CLAMP_TO`；旧逻辑对网格 apply 旋转缩放并隐藏曲线，导致叶子相对树干错位。
24. **取景/导出**：背景平面与离群体仍只排除出 bbox 与 GLB 选中集（`hide_render` 避免固定图拍到地板）；作者相机/灯光不参与渲染。
25. **包围盒**：`scene_bbox` 使用 depsgraph 求值后包围盒，含修饰器变形，避免曲线树干取景塌缩。
26. **瘦高取景**：自动选 front 时，若物体明显偏高且最大面是顶/底，改用最大侧面，避免椰子树库卡片变成俯视。
27. **双通道**：固定 PNG 与 `export_apply` 烘焙 GLB 均在依赖可用前提下生成，与 Blender 造型一致。
28. **存量**：全量 `refresh_blend_views.py`。

### BLEND 平面误剔（沙滩椅）

29. **根因**：座垫物体名为 `Plane`，旧规则「名称含 plane → 一律背景」将其剔除，固定图/GLB 只剩椅架。
30. **新规则**：名称像平面时，仅当尺寸 > 候选中位 ×8（过大）或极薄少面才当背景；同尺寸零件保留。无名极薄少面仍剔除。
31. **存量**：全量 `refresh_blend_views.py`。

### BLEND 程序材质 → GLB（沙滩椅座垫）

32. **根因**：座垫 Base Color 经 Wave/Gradient/ColorRamp 程序节点；EEVEE 固定图正常，glTF 不导出程序纹理 → 交互预览座垫发白。
33. **修法**：仅 `.blend` 且按需——Base Color 非图像链路时，Cycles Emission 烘成 1024 Base Color 贴图（无 UV 则 Smart UV），再 `export_apply` 导 GLB。
34. **存量**：全量 `refresh_blend_views.py`。

### 检索 Debug

35. **检索页 Debug 开关**：常态仅显示开关；开启后显示 N（噪声条数，默认 100，上限 5000）、M（探针数，默认 20）、「应用 N」「跑基准」与指标区。
36. **噪声向量**：仅写入 faiss（同维随机 L2 向量，预留 id 段），不创建 Asset；关闭 Debug 或后端启动时按 `faiss/debug_probe_ids.json` 清理残留。
37. **指标**：日常文字/以图检索返回 `elapsed_ms`；「跑基准」随机抽 ≤M 条真实视图做 hit@K 与耗时均值/中位/min/max。

### API

- 移除 `POST /api/assets/{asset_id}/reprocess`。
- `PUT /api/assets/{asset_id}/tags`：全量替换该资产标签列表并写回数据库（修改/删除后即时持久化）。
- `POST /api/assets/bulk-delete`
- `GET/POST /api/search/debug/status|enable|disable|benchmark`
